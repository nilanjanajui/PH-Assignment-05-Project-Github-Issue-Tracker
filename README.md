### 1. What is the difference between var, let, and const?
**Answer:**
`var`, `let`, `const` – All declare variables, but differ in scope and mutability. `var` is function-scoped and can be redeclared; `let` is block-scoped and can be reassigned; `const` is block-scoped and cannot be reassigned.


### 2. What is the spread operator (...)?
**Answer:**
**Spread operator** `...` - Expands elements of an array, string, or object into individual items, often used for copying, merging, or passing multiple values.


### 3. What is the difference between `map()`, `filter()`, and `forEach()`?
**Answer:**
`map()`, `filter()`, `forEach()` - Array methods: `map()` transforms each element into a new array, `filter()` creates a new array with elements that pass a condition, and `forEach()` executes a function on each element without returning a new array.


### 4. What is an arrow function?
**Answer:**
**Arrow function** - A compact function syntax using `=>` that automatically binds `this` from the surrounding scope and can return expressions implicitly.


### 5. What are template literals?
**Answer:**
**Template literals** - Strings wrapped in backticks (```) that support variable interpolation with `${...}` and multi-line text without special characters.
